# CodeStart IA

## Plataforma para aprender programación con ayuda de Inteligencia Artificial Generativa

CodeStart IA es una aplicación web académica desarrollada en Java Web con NetBeans. Su propósito es apoyar a estudiantes principiantes en el aprendizaje de programación mediante contenidos guiados, ejercicios prácticos, ejemplos de código y el uso de Inteligencia Artificial Generativa como herramienta de apoyo.

El proyecto no está enfocado únicamente en aprender sobre Inteligencia Artificial, sino en utilizar la IA Generativa como acompañamiento para aprender a programar, resolver dudas, comprender errores y mejorar la lógica de programación.

---

## Descripción del proyecto

Aprender a programar puede ser complicado para estudiantes que apenas comienzan, especialmente cuando se enfrentan a conceptos como variables, condicionales, ciclos, arreglos, funciones, clases, objetos, conexión a bases de datos o desarrollo web.

CodeStart IA busca atender esta necesidad mediante una plataforma educativa donde el usuario pueda consultar lecciones de programación, revisar ejemplos, recibir consejos de estudio y utilizar la IA Generativa como apoyo para comprender mejor los temas.

La plataforma está pensada como una herramienta académica que acompaña el proceso de aprendizaje, ayudando al estudiante a practicar y reforzar los conocimientos vistos en clase.

---

## Objetivo general

Desarrollar una plataforma web responsiva que apoye el aprendizaje de programación mediante lecciones, ejemplos, ejercicios y recomendaciones asistidas por Inteligencia Artificial Generativa, permitiendo a los usuarios registrar su progreso dentro del sistema.

---

## Objetivos específicos

- Presentar lecciones introductorias de programación.
- Mostrar ejemplos de código explicados de forma sencilla.
- Proponer ejercicios prácticos para reforzar el aprendizaje.
- Ofrecer recomendaciones para usar IA Generativa como apoyo al programar.
- Permitir el registro e inicio de sesión de usuarios.
- Registrar el progreso de aprendizaje de cada estudiante.
- Implementar un panel de administración para gestionar contenidos.
- Conectar la aplicación con una base de datos relacional.
- Publicar el proyecto en GitHub con documentación de instalación y ejecución.

---

## Módulos principales

### 1. Lecciones de programación

Este módulo contiene temas básicos para aprender a programar, como:

- Introducción a la programación.
- Variables y tipos de datos.
- Operadores.
- Condicionales.
- Ciclos.
- Arreglos.
- Métodos y funciones.
- Programación orientada a objetos.
- Introducción a Java Web.

### 2. Ejemplos de código

Este módulo muestra fragmentos de código con explicación paso a paso. El objetivo es que el estudiante no solo copie código, sino que entienda qué hace cada parte.

Ejemplo de contenidos:

- Programa Hola Mundo.
- Uso de variables.
- Condicional if/else.
- Ciclo for.
- Ciclo while.
- Clases y objetos.
- Conexión básica a base de datos.

### 3. Ejercicios prácticos

Este módulo permite consultar ejercicios para practicar los temas vistos en las lecciones.

Ejemplos:

- Crear una calculadora básica.
- Validar si un número es par o impar.
- Calcular el promedio de calificaciones.
- Registrar datos de un usuario.
- Crear una clase Alumno.
- Mostrar información desde una base de datos.

### 4. Ayuda con IA Generativa

Este módulo orienta al estudiante sobre cómo usar IA Generativa para aprender programación, por ejemplo:

- Pedir explicación de un error.
- Solicitar ejemplos similares.
- Pedir que se explique un código línea por línea.
- Generar ejercicios de práctica.
- Comparar distintas soluciones.
- Mejorar la lógica de un programa.

La IA no sustituye al estudiante, sino que funciona como una herramienta de apoyo para comprender mejor los temas.

### 5. Usuarios y progreso

Permite que el estudiante se registre, inicie sesión y marque lecciones o ejercicios como completados.

### 6. Administración

Permite al administrador agregar, editar y eliminar lecciones, ejemplos, ejercicios y recomendaciones.
