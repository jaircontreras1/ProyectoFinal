# FODA del proyecto

| Elemento | Descripcion |
|---|---|
| Fortalezas | Plataforma clara, responsiva, con contenidos organizados sobre IA Generativa, tips y normatividades. |
| Oportunidades | Puede crecer con evaluaciones, buscador, videos, material descargable, perfiles y analitica de progreso. |
| Debilidades | La primera version no incluye evaluacion automatica, chat, editor interactivo ni actualizacion automatica de normativas. |
| Amenazas | Existen plataformas educativas y herramientas de IA con mas recursos; las normas sobre IA cambian rapidamente. |

# FODA de la materia

| Elemento | Descripcion |
|---|---|
| Fortalezas | Permite aplicar planeacion, analisis, diseno, programacion web, bases de datos y documentacion. |
| Oportunidades | Desarrolla habilidades utiles para proyectos reales con Java Web y MySQL. |
| Debilidades | El tiempo puede limitar la profundidad de pruebas, diseno visual y seguridad avanzada. |
| Amenazas | Problemas de configuracion con NetBeans, Tomcat, XAMPP, conectores JDBC o versiones de Java. |

# Conclusion
El proyecto integra las fases principales del desarrollo de software y resuelve una necesidad educativa actual: orientar el uso responsable de la Inteligencia Artificial Generativa. La plataforma permite consultar lecciones, tips y normatividades, guardar progreso y administrar contenido basico. Como mejora futura puede incluir buscador, evaluaciones, roles mas completos y actualizacion de normativas desde fuentes oficiales.
